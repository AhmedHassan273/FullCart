Getting Started

Clone the Repository:

git clone https://github.com/AhmedHassan273/FullCart.git

Install Dependencies:

cd FullCart
npm install

Configure Environment Variables:

Create a .env file based on .env.example.

Set up your database connection, API keys, and other environment variables.

Run the Application:
npm start

Open in Browser: Open your browser and navigate to http://localhost:3000.
